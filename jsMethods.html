JS Methods

Definition

You can think of a method as a message that you send to a number, and the result that JavaScript gives you as the number's response to that message.

Code Example

> 123.456.toFixed(2);

123.45



>123.456.toPrecision(3);

123



>"cat".toUppercase()

CAT

Example
