JS Functions

Definition

A function is something that performs an action. Just like a method, a function can take an argument. The alert() function pops up a dialog box with the string that you pass in as an argument.

Code Example

>alert("Hello!");

undefined



>prompt("What is your favorite color?");

"purple"

To practice, open the JavaScript console in a Chrome browser and enter the code example above.
