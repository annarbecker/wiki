Inline Elements

Definition

Inline elements typically may only contain text and other inline elements. When rendered visually, inline elements do not usually begin on a new line.

Code Example

<!DOCTYPE html>

<html>

<head>

<title>Example Title</title>

</head>

<body>

<h1>This is a h1 Header</h1>

<p>Lorem ipsum <strong<dolor</strong< sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.</p>

</body>

</html>
