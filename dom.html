DOM Manipulation and Traversal

Definition

DOM is the Document Object MOdel. It provides a structured representation of the document (a tree) and it defines a way that the structure can be accessed from programs so that they can change the document structure, style and content. It can be manipulated in jQuery using different methods.

Code Example

$(document).ready(function(){

$("button#flowerButton").click(function(){

$("ul.unstyled").append('<img src="img/flower.jpg"> ');

});

});
