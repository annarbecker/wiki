JS Variables

Definition

JavaScript variables are containers for storing data values. You can set a variable equal to arrays, strings, numbers, etc.

Code Example

> var ourNames = ["Anna", "Brooke"];

undefined



> ourNames;

["Anna", "Brooke"]



>var randomNumber = 20;

undefined



>randomNumber + 30;

50



> var randomString = "Epicodus is fantastic!";

undefined



> randomString;

"Epicodus is fantastic!"

Example

To practice, open the JavaScript console in a Chrome browser and enter the code example above.
