Cascading Styles

Definition

This means that in your CSS stylesheet, a rule that is more specific will take precedence over less specific rules to the same element.

Code Example

p {

border: 5px solid yellow;

}

p.important {

border: 5px solid green;

}

p.more .important {

border: 5px solid blue;

}
